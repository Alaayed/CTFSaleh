### Arithmetic Mean
Use case: Include all the values, including outliers
Given a set of returns $R$ the arithmetic mean return ($AMR$) is: $$AMR = \frac{\sum_{i}^n R_i}{n}$$

### Geometric Mean
Use case: Compound the rate of returns over multiple periods
Given a set $R$ the geometric mean return ($GMR$) is :
$$GMR = \sqrt[n]{\prod_{i = 1}^n (1+R)}  \;\;- 1$$
### Harmonic Mean
Use case: Calculating the average share cost from periodic purchases in fixed money amounts
Give $N$ values of $X_i$ the harmonic mean($HM$) is:
$$ HM = \frac{N}{\sum_{i=1}^N \frac{1}{X_i}}$$
