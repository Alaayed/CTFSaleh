- **Gross Return**: The total return before deducting management and administration fees.
- **Net Return**: Return after management and administration fees are deducted.
- **Pretax Nominal Return**: Return before taxes. Each type of income maybe taxed at different rates
- **After-Tax Nominal Return**: Return after tax liabilities are deducted.
- **Real Return**: Nominal return after adjusting for inflation.
